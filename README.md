## Socket server

### Inicializar proyecto
```sh
mkdir websocket-dqn-server
cd websocket-dqn-server
npm init -y
```

### Instalar WebSocket
```sh
npm install ws
```

### Ejecutar servidor
```sh
node server.js
```
